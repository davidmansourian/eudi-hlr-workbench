Remote QES - Creating a Signature (EUDI Wallet used for authentication/authorisation)


High Level Description:
In this remote flow, the signature is created using a signing (private) key held within a Qualified Signature Creation Device operated by a Qualified Trust Service Provider.
It shall be highlighted that in this flow the signature is created by an external signing service which interacts with the Relying Party. The sole role of the EUDI Wallet in this flow is to be used for
authentication and authorisation purposes as part of the signature creation.
